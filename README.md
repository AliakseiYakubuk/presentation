# Self-presentation Page

## start project
 ```
 yarn start
 ```

## build project
 ```
 yarn build
 ```

## check types
 ```
 yarn ts
 ```

## check code style
 ```
 yarn eslint
 ```

## check styles
 ```
 yarn stylelint
 ```

## run all project checks 
 ```
 yarn lint
 ```

## preview project build
```
 yarn build
 yarn preview
```
