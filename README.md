# Self-presentation Page

## start project
 ```
 yarn start
 ```

## build project
 ```
 yarn build
 ```

## check types
 ```
 yarn ts
 ```

## lint project
 ```
 yarn lint
 ```

## lint code
 ```
 yarn eslint
 ```

## lint styles
 ```
 yarn stylelint
 ```

## preview project build
```
 yarn build
 yarn preview
```
